# Лабораторна робота №6: CI/CD pipeline
**Модель:** Математичне моделювання та аналіз струмів у електричних колах постійного струму за законами Кірхгофа (5 семестр).
**Автор:** Власійчук Данило, група АІ-235.
**Варіант:** Docker build + syntax check (парний номер).

## Опис
Цей проєкт містить автоматизований CI/CD pipeline, налаштований через GitHub Actions. 
При кожному пуші в гілку main виконуються наступні кроки:
1. Встановлення залежностей (Flask, NumPy).
2. Перевірка синтаксису файлу app.py.
3. Тестова збірка Docker-образу для перевірки контейнеризації.
